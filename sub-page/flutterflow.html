Flutter Flow

El desarrollo para dispositivos móviles está en su auge, pero, ¿cómo se hace?

{Mobile App}

  • Una aplicación móvil, o app, es un programa informático diseñado para funcionar en dispositivos móviles como teléfonos inteligentes y tabletas. Las apps se ejecutan en sistemas operativos como Android o iOS y pueden tener una gran variedad de propósitos, desde entretenimiento hasta finanzas, salud o educación.

{Types}

  • {Native}
    Plataformas específicas con lenguajes determinados; aprovechan al máximo el hardware; y se descargan desde las tiendas oficiales. Pasa por el proceso de tienda.
  • {PWA}
    Construidas con HTML5, JavaScript, y CSS; el usuario final tendrá la misma experiencia y usabilidad que una aplicación nativa; deben cumplir con ciertos estándares para considerarse PWA. Otra forma de verlo, son páginas web que pueden ser agregadas a la pantalla de inicio como si fueran una “aplicación” normal. No pasa necesariamente por el proceso de tiendo.
  • {Hybrid}
    Se construyen con frameworks o tecnologías que permiten un solo desarrollo en un lenguaje y sean capaces de ejecutarse en cualquier plataforma. Pasa por el proceso de tienda.

{Flutter Flow}

  • Es una herramienta de desarrollo móbil basada en Flutter, que junto a Firebase, son las herramientas más utilizadas para el desarrollo de aplicaciones móviles. Flutter Flow es una herramienta visual que permite a los desarrolladores crear aplicaciones móviles sin necesidad de escribir código.
