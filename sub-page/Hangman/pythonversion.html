Hangman

Pero ahora es la versión creada en python. Revisemos las diferencias que hay entre los lenguajes de programación.

{Concept}

  • Python es un lenguaje de programación de alto nivel, el cual hace especial énfasis en ser fácil de leer y escribir. Desde 1991, ha sido uno de los lenguajes más populares para la creación de códigos tanto para principiantes como para expertos que aprovechan sus facilidades.

{Differences}

  • Hay una múltitud de diferencias entre un lenguaje como C++ y Python, comenzando desde la propia fecha de creación, siendo C++ un lenguaje más antiguo, pero enfoquemonos en algunos apartados específicos.
  • {Sintaxis}
    Python cuenta con una sintaxis más sencilla y fácil de leer, mientras que C++ es más complejo y detallado. Esto ayuda al momento de escribir y leer, pero puede ser un problema al momento de depurar.
  • {Compilation}
    Python es un lenguaje interpretado, lo que significa que no necesita ser compilado para ser ejecutado, mientras que C++ es un lenguaje compilado, lo que significa que necesita ser compilado antes de ser ejecutado.
  • {Performance}
    C++ es un lenguaje más rápido que Python, ya que es un lenguaje compilado y Python es un lenguaje interpretado, además de que tiene algunas instrucciones que pueden llegar a ser muy abstractas. Esto significa que C++ es más rápido en la ejecución de programas.
